# GrowLogic

🌱 GrowLogic

GrowLogic — интеллектуальная система управления теплицей, разработанная для автоматизации процессов выращивания растений с использованием технологий Интернета вещей (IoT).

Система осуществляет непрерывный мониторинг параметров окружающей среды, автоматически управляет оборудованием теплицы и предоставляет пользователю возможность удаленного контроля через удобный веб-интерфейс.

⸻

🚀 Возможности

* 🌡️ Мониторинг температуры воздуха
* 💧 Контроль влажности почвы
* 💨 Измерение влажности воздуха
* ☀️ Контроль уровня освещенности
* 🚿 Автоматическое управление поливом
* 🌀 Управление вентиляцией
* 💡 Автоматическое управление освещением
* 📊 Отображение данных в режиме реального времени
* 📈 Хранение истории измерений
* 🔔 Уведомления при выходе параметров за допустимые пределы
* 🌐 Удаленное управление через веб-интерфейс

⸻

🛠 Используемые технологии

Аппаратная часть

* ESP32
* Датчик температуры и влажности
* Датчик влажности почвы
* Датчик освещенности
* Релейный модуль
* Насос для полива
* Вентилятор
* Светодиодное освещение

Программная часть

* C++
* Arduino Framework
* HTML
* CSS
* JavaScript
* REST API
* WebSocket (при наличии)
* MQTT (при наличии)

⸻

⚙️ Основные функции

* автоматический контроль микроклимата;
* интеллектуальное управление исполнительными устройствами;
* мониторинг показателей в режиме реального времени;
* удаленное управление системой;
* сохранение статистики работы;
* повышение эффективности выращивания растений;

Создание интеллектуальной системы управления теплицей, способной автоматически поддерживать оптимальные условия для выращивания растений, минимизируя участие человека и повышая эффективность использования ресурсов.

⸻

🔮 Планы по развитию

* интеграция с Telegram-ботом;
